# InkDAQ

## Introduction

InkDAQ is a system that acquires telemetry data from a pen, in order to assess one's handwritig.

At the moment, the project is in very early stages, and will only measure the writing force. Sensors to measure data such as grip force, or an accelerometer (to measure one's writing speed) might be added in the future.

It is designed to be a modular system, with a custom-designed external data acquisition device, while the pen will act as a passive sensor probe, connected via a cable to the DAQ device, to allow for a more modular approach as the pen can be changed.
